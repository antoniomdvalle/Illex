# Illex
## The Intelligent Bank

Illex is a **data-driven financial management system** designed to help users understand and grow their personal wealth.

Instead of relying on opinions or generic financial advice, Illex focuses on **structured financial data**, allowing users to clearly visualize their financial situation and make better decisions.

---

# About the Project

Illex was created with the idea of **centralizing personal financial information** and transforming raw financial data into useful insights.

The system allows users to:

- Track income and expenses
- Monitor their financial balance
- Create and manage financial goals
- Visualize financial summaries through dashboards
- Generate reports based on their financial activity

The long-term goal of Illex is to evolve into a **data-driven financial intelligence platform**, capable of analyzing financial behavior and suggesting better financial strategies.

---

# Core Features

## Authentication

- Secure user authentication  
- JWT-based login system  

---

## Financial Records

Users can register financial transactions including:

- Type (income or expense)
- Value
- Date
- Category
- Description

---

## Financial Goals

Users can create and manage goals categorized as:

- Short-term (up to 1 year)
- Medium-term (1–5 years)
- Long-term (5+ years)

---

## Dashboard

The dashboard provides a quick overview of the user's financial situation:

- Total income
- Total expenses
- Current balance
- Active financial goals

---

## Reports

Generation of financial reports based on stored data.

Examples:

- Monthly summaries
- Financial activity reports
- Data exports
